# 🎙️ Wave2MD

> Transforma arquivos de áudio em documentos estruturados para iniciar projetos de desenvolvimento — PRD, README, TODO, tudo pronto para **Vibe Coding**.

## O que é

Wave2MD é uma aplicação PWA que converte gravações de voz em outputs estruturados prontos para desenvolvimento:

- 📄 **PRD** (Product Requirements Document)
- 📖 **README** completo
- ✅ **TODO list** priorizada
- 🗂️ **Estrutura de pastas** sugerida

## Stack

- **Frontend:** React 19 + TypeScript + Vite
- **Estilização:** CSS Modules / Tailwind
- **Audio:** Web Audio API
- **AI:** Integração com modelos de transcrição e geração

## Como usar

```bash
npm install
npm run dev
```
